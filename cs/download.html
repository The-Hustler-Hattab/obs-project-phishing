Vydání pro Linux je oficiálně dostupné pro Ubuntu 14.04 a novější. Je nutné mít nainstalovaný FFmpeg.

Pro Ubuntu 14.04 LTS, FFmpeg není oficiálně k dispozici, takže budete potřebovat speciální PPA:
sudo add-apt-repository ppa:kirillshkrogalev/ffmpeg-next
sudo apt-get update && sudo apt-get install ffmpeg

Pro Ubuntu 15.04 a následující verze je již FFmpeg oficiálně dostupný:
sudo apt-get install ffmpeg

Po instalaci FFmpeg, nainstalujte OBS Studio pomocí:
sudo add-apt-repository ppa:obsproject/obs-studio
sudo apt update
sudo apt-get update && sudo apt-get install obs-studio
