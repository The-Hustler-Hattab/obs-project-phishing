obs-websocket - Remote-control OBS Studio using WebSockets 5.0.1

Only download obs-websocket from here if you are running OBS v27. For v28 and above, obs-websocket is included with OBS Studio.

A WebSocket RPC API for OBS Studio. The WebSocket server runs on port 4455. See here for client software and libraries that support obs-websocket's protocol.
