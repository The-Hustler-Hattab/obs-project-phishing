OBS Studio Kawaii Edition 1.3.2

Download
  • Author Monsteer
  • Creation date Apr 5, 2022
Overview Updates (7) Reviews (2) History Discussion
A colorful and funny Kawaii styled theme to make the stream time more relaxed.

Give it a try!

Icons have been added for the main docks and for some of the most used plugins.
Special fixes for some plugins to display correctly.

This theme is still a WIP, but is fully functional.
It's very hard to use pinky colors in an OBS theme but... why not?

Recommended resolution 1920x1080.

Thanks to all the other themes that helped me understand how to code this theme, without them I couldn't have created it.

Thank you very much for all your incredible work!

How to install

1. Unpack files into your OBS Studio Theme Folder (...\obs-studio\data\obs-studio\themes)
2. Restart OBS Studio
3. Go to Settings > General and select the theme
